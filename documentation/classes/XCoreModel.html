XCoreModel

ultra fetter unterbau des models, alta!

in fact, this class does everything about database access. it connects to a running mysql database. it requires having mysqli installed and enabled for php

it can find, update, save and delete data for finding, there is a nice functionality that allows you providing an array filled with conditions and constraits but you can provide just a mysql query string as well as the find method does not provide the whole mysql query functionality it is built up as a singleton class so yout can get access to this class via the public static function getInstance
